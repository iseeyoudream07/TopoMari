设置 / 通用

通用设置

配置公开面板的视觉主题、自动日夜模式和 GeoIP 定位。

视觉主题

保留 TopoMari 原版，也可以切换到原生适配的毛玻璃主题。 查看主题来源

自定义配色

分别设置日间和夜间的背景色与强调色；保存后公开面板会立即使用。

主题自动切换

按北京地理坐标每日计算日出和日落，日出后使用日间模式，日落后使用夜间模式。

链路健康状态阈值

为所有链路设置统一的关注与异常阈值；链路编辑器中的单独设置仍可覆盖这些值。

关注状态 延迟达到阈值，或丢包率超过阈值时，链路显示为关注。
异常状态 延迟或丢包率达到更高阈值时，链路显示为异常。

无数据时仍显示未知，最近一次探测失败时仍直接显示失败。

GeoIP 数据库

通过 Komari API 密钥读取节点 IP，并使用 MaxMind 国家数据库定位；IP 和密钥只在服务端处理，不会发送到公开面板。

地理位置数据提供商TopoMari 通过 Komari 的本地 GeoLite2 Country 数据库查询。 MaxMind
更新 GeoIP 数据库正在检查 Komari GeoIP 状态…

主题设置

主题设置

配置地球动效，并精细调整 Glassmorphism 的背景媒体、毛玻璃强度、边框和圆角。

Glassmorphism 专属设置 请先在“设置 → 通用”中切换并保存 Glassmorphism。TopoMari 原版不接受自定义背景或玻璃表面修改。

自定义背景

支持亮暗模式分别使用图片或视频。远程地址会由访客浏览器直接加载，建议使用 HTTPS。

亮色模式背景
暗色模式背景

背景效果

模糊只作用于背景媒体；正遮罩使背景变暗，负值降低背景透明度。

毛玻璃卡片

这些设置会作用于公开面板的统计、拓扑、链路与节点卡片，也同步用于后台预览。

SETTINGS / SITE

站点设置

修改公开面板的名称、页面描述和浏览器图标。

站点信息

名称会同步到浏览器标题和公开面板标题；描述用于页面元信息。

Komari API 密钥

用于服务端读取节点地址和管理 GeoIP。密钥保存在持久化 data 目录，浏览器只能看到配置状态，无法读取原值。

保存后输入框会立即清空；以后填写新值将替换现有密钥。

自定义 Favicon

支持 PNG 或 ICO，最大 2 MiB；文件保存在持久化 data 目录。
